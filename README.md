# HyperGrowth Resource Pack

**HyperGrowth** is a Minecraft resource pack designed to make farming easier and more intuitive by displaying the growth stages of crops directly above the plants. With HyperGrowth, you can instantly see which crops are fully grown and which are still developing, helping you plan your harvest more efficiently.

## Features

- Displays growth stages above all standard crops in Minecraft using color.
- Fully compatible with wheat, carrots, potatoes, beetroot, and more.
- Includes support for additional crops, such as **Amethyst**.
- Clear visual indicators show whether crops are fully grown or still growing.
- Lightweight for smooth gameplay.

## Installation

### Modrinth Website
1. Download the latest version of HyperGrowth from [Modrinth](https://modrinth.com/project/hypergrowth) by clicking **Download**.
2. Move the `.zip` file to your Minecraft `resourcepacks` folder.
   - **Windows:** `%appdata%\.minecraft\resourcepacks`
   - **Mac:** `~/Library/Application Support/minecraft/resourcepacks`
   - **Linux:** `~/.minecraft/resourcepacks`
3. Open Minecraft, go to **Options → Resource Packs**, and activate **HyperGrowth**.
4. Enjoy a smarter farming experience!

### Modrinth App
1. Open the Modrinth app and go to your Minecraft instance.
2. Click **Install content** and, under the resource packs tab, search for **HyperGrowth**.
3. Click **Install**.
4. Enjoy a smarter farming experience!
